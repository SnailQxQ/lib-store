# Lib-store
前后端分离的网盘项目  基于SpringBoot + MySql + FFmpeg + VUE 

## &#x1F449;现有功能 
  + 资源管理（上传、下载、移动、修改、删除）
  + 资源分享
  + 资源共享
  + 资源转存
  + 资源预览（pdf编辑、图片、视频）（office 三件套待支持）
  + 断点续传
  + 敏感词过滤
  + 流式视频播放
  + 秒传支持
