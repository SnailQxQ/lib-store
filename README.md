# Lib-store
前后端分离的网盘项目 后台基于SpringBoot SSM + MySql + FFmpeg 

现有功能 
  资源管理、资源分享、资源共享、资源转存、资源预览、断点续传、敏感词过滤、视频在线播放、秒传支持
